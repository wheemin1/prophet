# 한줄신탁 v2 — 화면 구성 정리

## A. 상단(Hero)

1. **타이틀**: `한줄신탁`
2. **부제**: `예언은 오늘 단 하나만 주어진다`
3. **목소리 칩(작게)**: `오늘의 목소리: 균형 (자동)`

   * 툴팁: `설정에서 한 번만 바꿀 수 있어요.`
4. **헤더 우측 아이콘**: \[설정] \[정보] (아이콘만, 텍스트 툴팁)

> 주의: 헤더에 버튼이 많지 않게. \*\*CTA는 아래 ‘봉인’\*\*에 집중.

---

## B. 개인화 입력(접을 수 있는 카드)

* **카드 제목**: `개인화 입력` (좌측에 열림/닫힘 아이콘)
* **설명 한 줄(짧게)**: `입력 정보는 서버에 저장되지 않으며, 예언 생성 시드로만 사용됩니다.`
* **필드**: 이름(선택), 생년월일(선택)
* **버튼**: `개인화 적용`
* **적용 후**: 카드는 자동으로 접고, **헤더 아래에 ‘프로필 칩’** 노출

  * 예: `홍길동 · 1994-01-07` / 변경 클릭 시 다시 펼침

> 이유: 첫 화면의 복잡도를 줄이고, 봉인 의식 집중도를 높임.

---

## C. 네비게이션 탭

* `오늘 | 이주 | 이달 | 올해`
* 활성 탭만 강조(밝은 배경/하이라이트).
* 탭 전환 시 **타이머 문구도 해당 레벨로 동기화**.

---

## D. 다음 갱신(절대시간)

* 라벨 통일: `다음 예언`
* 값: `오늘 23:59 (Asia/Seoul)` 혹은 `내일 00:00 (Asia/Seoul)` (중 하나만 사용)
* 이달/올해 탭에서는:

  * 이달: `다음 예언: 다음 달 1일 00:00 (Asia/Seoul)`
  * 올해: `다음 예언: 내년 1월 1일 00:00 (Asia/Seoul)`

> 지금 스크린샷에 “내일 00:00”과 “다음 예언” 표기가 혼재 → **한 표현으로 고정**.

---

## E. 봉인 의식(메인 CTA)

* 타이틀: `봉인 의식`
* 설명(2줄 고정, 줄바꿈 동일):

  * `룬을 3초간 길게 누르면 예언이 공개됩니다.`
  * `데스크톱: 스페이스바 3초 홀드`
* **룬 원형**: 진행 링 + 중심 심볼. 손/커서 이탈 시 0.4초 유예 후 리셋.
* **광고 정책**: 전면(비네트/인터스티셜)은 **첫 해제 중 50\~100% 구간에서 최대 1회/일**.

---

## F. 예언 공개(결과 카드)

* 섹션 제목: `오늘의 예언` (탭에 따라 `이주의 예언/이달의 예언/올해의 예언`)
* 본문(큰 글자, 1문장): 예: `급할수록 저장을 누르라.`
* 하단 버튼:

  * \[공유] → Web Share API / 미지원 시 URL 복사
  * \[저장] → PNG 카드(캔버스) 다운로드
* 섹션 구분선 아래(2차 액션):

  * \[👍 좋았어] \[🤔 미묘해] — 라벨 아래 미세 문구: `당신의 반응은 브라우저에만 저장됩니다.`
* (선택) 오른쪽 하단 작은 `시그닛 코드` 배지: `코드 ST-8G2KQ`

---

## G. 내 예언 기록

* **빈 상태 문구**(짧게):

  * 제목: `아직 예언 기록이 없습니다.`
  * 보조: `첫 예언을 받아보세요!`
* **목록 카드**(가로):

  * 왼쪽: 날짜/레벨 배지(`오늘/이주/이달/올해`)
  * 오른쪽: 예언 문장 1줄 미리보기
  * 클릭 시 상세 카드 오픈(+ 공유/저장 가능)

---

## H. 설정 & 데이터 관리

* **설정** (토글 3개만 상단 고정)

  * `사운드 효과` / `모션 효과` / `다크 모드`
* **데이터 관리**

  * \[내보내기(JSON)] \[가져오기] \[초기화]
  * 초기화 버튼은 **색상 경고 + 2단계 확인**
* **법적 고지**(아주 작게)

  * `엔터테인먼트 목적이며, 의료·금전 조언이 아닙니다.`
  * `모든 데이터는 브라우저에만 저장되며 서버로 전송되지 않습니다.`
  * \[개인정보처리방침] \[이용약관]

---

## I. 광고 위치(웹)

* **앵커 배너**: 화면 하단 고정(고정 높이 미리 예약)
* **인피드/네이티브**: `내 예언 기록` 위쪽 1개
* 전면 형식(비네트): 페이지 전환 또는 첫 봉인 시 **1회/일**
* CLS 방지: 광고 컨테이너는 항상 자리 예약

# 마이크로카피 교정(스크린샷 기준)

* 헤더 칩: `오늘의 목소리: 균형 (자동)`
* 개인화 카드 설명: `입력 정보는 서버에 저장되지 않으며, 예언 생성 시드로만 사용됩니다.`
* 봉인 의식 설명:

  * `룬을 3초간 길게 누르면 예언이 공개됩니다.`
  * `데스크톱: 스페이스바 3초 홀드`
* 타이머: `다음 예언: 내일 00:00 (Asia/Seoul)` ← 하나로 통일
* 결과 카드 제목: `오늘의 예언`
* 공유 버튼: `공유` / 저장 버튼: `저장`
* 피드백 안내: `당신의 반응은 브라우저에만 저장됩니다.`
* 기록 빈 상태: `아직 예언 기록이 없습니다. 첫 예언을 받아보세요!`

# 시각/레이아웃 정돈

1. **여백 스케일**

   * 섹션 간 상단 32px / 내부 16px / 컴포넌트 8px
   * 카드 모서리 동일(rounded-2xl), 그림자 깊이 1\~2단계로 통일
2. **타이포 스케일**

   * H1 28–32, 섹션제목 18–20, 본문 15–16, 보조 13–14
   * 예언 본문은 **한 단계 더 큼**(20–22) + 자간 약간 넓힘
3. **색/강조**

   * 금색 하이라이트는 **제목·버튼·룬 발광**에만 사용(남용 방지)
4. **경계선**

   * 카드 보더는 반투명 1px, 내부 구분선은 더 옅게
5. **아이콘**

   * 탭/버튼 아이콘 크기 18–20, 텍스트와 수직정렬 맞추기

# 상태/전환 정의

* **첫 방문**: 개인화 카드 **열림**, 기록 **빈 상태**, 봉인 전 안내 강조
* **개인화 적용 후**: 카드 자동 **접힘**, 헤더에 프로필 칩 표시
* **봉인 진행 중**: 버튼·탭 비활성화(포커스 락), 진행률 0→100%
* **예언 공개 후**: 봉인 컴포넌트는 **축소 상태**로 접기(결과 카드 집중)
* **다음 날 재방문**: 해당 날짜 예언 바로 표시(재생성 없음) + 타이머 업데이트
